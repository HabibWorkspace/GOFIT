## 📦 What's Included

### Core Files (Copy to C:\gymapp\)
- `run.py` - Production server
- `config_local.py` - Local configuration
- `requirements_local.txt` - Dependencies (rename to requirements.txt)
- `.env.local` - Environment template (rename to .env)
- `config_bridge_local.txt` - Bridge config (rename to config.txt)
- `start_app.bat` - Manual start script

### Data Migration
- `export_data.py` - Export from PythonAnywhere
- `import_data.py` - Import to local
- `verify_data.py` - Verify data
- `regenerate_qr_codes.py` - Update QR codes
- `update_app_for_local.py` - Update imports

### Scripts (Copy to C:\gymapp\scripts\)
- `install_services.bat` - Install all services
- `check_all.bat` - Check status
- `restart_all.bat` - Restart services
- `power_settings.bat` - Configure PC
